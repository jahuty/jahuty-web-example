Jahuty beta

Welcome to Jahuty's "web" example using a browser-based script.

This application includes the following examples:

  • Rendering a snippet
  • Rendering a snippet with parameters
  • Rendering a snippet with the latest content

These examples utilize the following "system test" setup:

  • The "system test" organization exists in Jahuty with an API key.
  • The organization has three snippets: 1, a static phrase; 62, a phrase with two variables, foo and bar; and, 102, a snippet with staged changes.
